## Losslessly convert these image types to WebP using Img2WebP:
- PNG (*.png)
- APNG (*.png)
- JPEG 1 (*.jpeg, *.jpg, *.jpe, *.jfif, *.exif)
- JPEG XR (*.jxr, *.wdp, *.wmp)
- JPEG XL (*.jxl)
- BMP (*.bmp, *.dib, *.rle)
- TIFF (*.tiff, *.tif)
- TGA (*.tga)
- GIF (*.gif)
- AV1/AVIF (*.avif)
- HEIC (*.heic)
